# Form Flow

A modern, feature-rich form builder application built with React, Vite, and Tailwind CSS. Create, manage, and export forms with ease.

## Features

✨ **Key Capabilities:**
- **Intuitive Form Builder** - Create custom forms with a drag-and-drop interface
- **Multiple Export Formats** - Export form data to Excel (XLSX) and other formats
- **Real-time Preview** - See your form changes instantly
- **State Management** - Efficient state management with Zustand
- **Firebase Integration** - Cloud-based backend for data persistence
- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Notifications** - User-friendly toast notifications with React Hot Toast
- **Routing** - Navigate between different views and pages
- **Code Quality** - ESLint configuration for maintaining code standards

## Tech Stack

- **Frontend Framework:** React 19.2.6
- **Build Tool:** Vite 8.0.12
- **Styling:** Tailwind CSS 4.3.0
- **State Management:** Zustand 5.0.14
- **Routing:** React Router DOM 7.15.1
- **Backend:** Firebase 12.14.0
- **Data Export:** XLSX 0.18.5, File Saver 2.0.5
- **Notifications:** React Hot Toast 2.6.0
- **Code Quality:** ESLint 10.3.0
